# AI Web3 School Cohort 0

公开学习仓库，用于记录 AI x Web3 学习过程、每日打卡、Hackathon 项目想法与实验代码。

## 当前方向

Hackathon 项目：AI 辅助 Web3 学习助手。

## 学习目标

用 21 天完成一个可展示的 AI x Web3 Hackathon MVP，并用 GitHub 记录每天的 proof-of-work。

## 重要文件

- `profile.md`：个人学习画像
- `learning-plan.md`：当前学习计划入口
- `notes/learning-roadmap.md`：阶段路线
- `notes/21-day-daily-plan.md`：21 天每日任务
- `daily/`：每日学习打卡
- `tasks/`：每阶段任务记录
- `experiments/`：小实验与 Demo 代码
- `hackathon/`：项目想法、问题定义、Pitch 与提交材料
- `handbook-feedback/`：Handbook 学习反馈
- `submissions/`：最终提交材料
- `templates/`：常用模板
