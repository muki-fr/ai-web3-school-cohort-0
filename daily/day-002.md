5. 运行 `git push`，推送到 GitHub

## 命令理解

- `git status`：查看当前仓库有哪些文件被修改、哪些文件还没有提交
- `git add .`：把当前目录下的改动加入暂存区，准备提交
- `git commit -m "message"`：把暂存区内容保存成一次 Git 提交
- `git push`：把本地提交推送到 GitHub 远程仓库
